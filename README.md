# csgo-gc-installer
Installer for CS:GO GC by GT-610 (it will install own version of CS:GO, that have much more features than CS:GO in Steam)

*It's not required for CS:GO GC patch!*
1. it also downloads gc-inventory-changer
2. it's for windows only
3. it also have csgolauncher if you want to every time selecting between inventory changer and game itself without opening game folder (good for steam experience)
4. it also will have added patch updates (in csgolauncher)
